<p>This app aims to make Backups on Android easy and fast using <a href='https://restic.net/' target='_blank' rel='nofollow'>Restic</a>. It also makes use of <a href='https://github.com/termux/proot' target='_blank' rel='nofollow'>termux/proot</a> to run the Restic linux binaries on Android. This project is still in a very early state and contributions are welcome!</p><p><b>Disclaimer:</b> This project is not an official app made by the restic team.</p><p><br><b>Features:</b></p><ul><li>Manage Restic Repositories (S3 is currently the only supported protocol)</li><li>Manage Restic Snapshots</li><li>Manage Folders for Backup</li><li>Configure Schedules for automatic Backups</li><li>Configure Cleanup Policies for Folders</li><li>Progress Notification</li></ul>
